Welcome to the Weather Data API

Get historical temperature data from various weather stations across Europe.

How to Use the API

URL format:

http://127.0.0.1:5000/api/v1/station/date

where:

  • station: The station ID (see available stations below)
  • date: The date in YYYYMMDD format

Example:

http://127.0.0.1:5000/api/v1/10/19981025
