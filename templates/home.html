Welcome to the Weather Data API

Get historical temperature data from various weather stations across Europe.

1. Get temperature for specific date

http://127.0.0.1:5000/api/v1/station/date
  • station: Station ID number
  • date: Date in YYYYMMDD format
Example: http://127.0.0.1:5000/api/v1/10/19981025

2. Get all data for a station

http://127.0.0.1:5000/api/v1/station
  • station: Station ID number
Example: http://127.0.0.1:5000/api/v1/10

3. Get yearly data for a station

http://127.0.0.1:5000/api/v1/yearly/station/year
  • station: Station ID number
  • year: Year in YYYY format
Example: http://127.0.0.1:5000/api/v1/yearly/10/1998
